## 改写前后原文对照

### 原文（tests/test_data/article_health.md）

```
# 幽门螺杆菌阳性怎么办？

幽门螺杆菌是一种寄生在胃里的细菌，在中国感染率约 40%-60%。
世界卫生组织把它列为 I 类致癌物，和吸烟、酒精同级。
但这不是说感染了就一定得胃癌。

从感染到胃癌需要经过 5 个阶段：HP 感染 → 慢性胃炎 → 萎缩性胃炎
→ 肠化生 → 异型增生 → 胃癌。这个过程可能持续几十年。
...
```

### 改写后（示例关键段落）

```
# 幽门螺杆菌阳性怎么办？先别慌，三类人需要治

> 先说结论：幽门螺杆菌阳性 ≠ 得胃癌。是否需要治疗，取决于
> 你有没有症状、胃镜结果如何、以及有没有胃癌家族史。

## 感染了会怎样？

根据 WHO 国际癌症研究机构（IARC）的分类，幽门螺杆菌属于
I 类致癌物——和吸烟、酒精同级（Lancet Gastroenterol Hepatol, 2023）。
但这不意味着感染了就一定会得胃癌。

从感染到癌变的五阶段路径：
1. HP 感染
2. 慢性胃炎（大多数感染者停留于此）
3. 萎缩性胃炎
4. 肠化生
5. 异型增生 → 胃癌
...
```
