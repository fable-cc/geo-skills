# 🛰️ GEO 月报演示

> 模拟数据 · 仅供演示 · 非真实客户数据

这是用 **景一 GEO 监测平台 V10** 自动生成的企业 AI 搜索引擎可见度月报。

## 月报包含 9 大模块

| 节 | 内容 |
|----|------|
| 一 | 月度总览 — AI 可见度总分 + 覆盖率 + 引用率 KPI |
| 二 | AI 平台可见度详情 — DeepSeek/豆包/Kimi/通义千问/文心一言/元宝 6 平台 |
| 三 | 核心问题覆盖详情 — 每个监测问题的排名和竞品 |
| 四 | SVG 图表 — 引擎覆盖率柱状图 + 历史趋势折线图 + 竞品对比 |
| 五 | 竞品动态 — 竞品出现频次和威胁等级 |
| 六 | 内容资产状态 — 各平台覆盖情况 |
| 七 | 结构化数据状态 — llms.txt/FAQ Schema 等检查 |
| 八 | 行动建议 — 🚨紧急 / 🟡重要 / 🟢建议 三级 |
| 九 | 距 22.4% 引用率临界点追踪 + 预估到达时间 |
