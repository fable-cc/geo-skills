# 한국어 GEO 리라이트 프롬프트

[`geo-rewrite-prompt-ko.md`](../../prompts/geo-rewrite-prompt-ko.md)의 System Prompt를 복사하여 원하는 LLM에 붙여넣어 사용할 수 있습니다.

## 지원 플랫폼

| 플랫폼 | 특징 |
|---|---|
| 지식인 / 네이버 블로그 | 질문형 도입부, 개인 경험, '도움이 되었어요' 유도 |
| Brunch / Tistory | 에세이형 도입, 구독 유도, 카테고리 태그 |
| Google Discover | 첫 150자 핵심 응축, 매력적인 숫자 |
| Instagram | 시각적 구분, 이모지 글머리, 해시태그 제안 |

## 다섯 가지 차원

1. **Q&A 구조화** — 글 첫머리에 3~5개의 질문과 답변
2. **엔티티 임베딩** — 고유명사 + 최초 설명 + 수치 데이터
3. **인용 앵커** — 3~5곳의 검증 가능한 인용
4. **구조화 마크업** — 엄격한 제목 계층 + 리스트 + 굵은 글씨
5. **플랫폼 적응** — 배포처에 따른 조정

## 전체 버전

프로젝트 루트의 [`prompts/geo-rewrite-prompt-ko.md`](../../prompts/geo-rewrite-prompt-ko.md)를 참조하세요.
