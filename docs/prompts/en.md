## Supported Platforms

| Platform | Adaptation |
|----------|-----------|
| ChatGPT Search | Q&A structure, citation anchors, authoritative tone |
| Perplexity | Dense entities, numbered lists, summary section |
| Gemini | Structured markup, FAQ section, data tables |
| Quora | Personal experience, question-first, actionable takeaways |
| Reddit | Subreddit-aware tone, TL;DR, community references |
| Substack | Newsletter format, subscription CTAs, serialized structure |
| Medium | Narrative lead, curated tags, reading time estimate |
| Google Discover | First 150 chars condensed, compelling numbers, alt text |
| Instagram / Pinterest | Visual breaks, emoji bullets, hashtag suggestions |

## Five Dimensions

1. **Q&A Structuring** — 3-5 H2 questions with immediate answers
2. **Entity Embedding** — Named entities + first-occurrence definitions
3. **Citation Anchoring** — 3-5 verifiable authoritative citations
4. **Structural Markup** — Strict heading hierarchy + lists/tables + bold keywords
5. **Platform Adaptation** — Tone and structure per platform
