# 日本語版 GEO リライトプロンプト

[`geo-rewrite-prompt-ja.md`](../../prompts/geo-rewrite-prompt-ja.md) の System Prompt をコピーし、任意の LLM に貼り付けて使用できます。

## 対応プラットフォーム

| プラットフォーム | 特徴 |
|---|---|
| Yahoo!知恵袋 / Note | 質問型の冒頭、個人体験、「参考になった」誘発 |
| note | 絵文字見出し、マガジン連載、一人称文体 |
| Google Discover | 先頭 150 字に核心、数字を含む |
| Instagram / Pinterest | 視覚的区切り、箇条書き + 絵文字、ハッシュタグ提案 |

## 五つの次元

1. **Q&A 構造化** — 記事冒頭に 3〜5 の質問と回答
2. **エンティティ埋め込み** — 固有名詞 + 初出時の説明 + 数値データ
3. **引用アンカー** — 3〜5 か所の検証可能な引用
4. **構造化マークアップ** — 厳格な見出し階層 + リスト + 太字
5. **プラットフォーム適応** — 配信先に応じた調整
