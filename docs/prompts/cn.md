# 中文版 GEO 改写提示词

复制 [geo-rewrite-prompt.md](../../geo-rewrite-prompt.md) 中的 System Prompt，粘贴给任何 LLM，即可获得 GEO 优化版的文章。

## 适用平台

| 平台 | 关键词 |
|------|--------|
| 知乎 | 问答型开头、个人经验、专业背书 |
| 微信公众号 | 叙事导入、分段清晰、订阅引导 |
| 百家号 | 前 150 字浓缩核心、SEO 友好 |
| 小红书 | 视觉化分节、表情符号、标签优化 |

## 五维规则

1. **问答结构化** — 文章开头放 3-5 个 H2 问题 + 直接回答
2. **实体植入** — 高密度专有名词 + 首次解释 + 数据支撑
3. **引用锚点** — 每篇 3-5 处可验证的权威引用
4. **结构化标记** — 严格标题层级 + 列表表格 + 关键词加粗
5. **平台适配** — 按平台差异调整语调和结构

## 直接使用

```bash
# 复制提示词
cat geo-rewrite-prompt.md

# 或使用命令行工具
python3 geo_rewrite.py --input 我的文章.md --platform zhihu
```
